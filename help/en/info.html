VidSubRenamer

The Problem

When downloading TV series seasons and subtitle packs from different sources, you often encounter a mismatch in file names.

For example:

  • Video: SerieName.S01E01.HDTV.x264.mp4
  • Subtitle: Name.S01E01.WEB.x264.srt

This mismatch leads to issues with automatic subtitle loading.

The Solution

VidSubRenamer automates the renaming process, ensuring that video files and their corresponding subtitles have identical names, eliminating the need for manual renaming. Each name must include a tag for the season and episode in the format SxxEyy.

How does it work?

  1. Copy both the video and subtitle files into a folder.
  2. In VidSubRenamer, select the folder.
  3. Choose the existing file extensions for the videos and subtitles.
  4. Decide the renaming direction (video → subtitle or subtitle → video).
  5. Let VidSubRenamer do the rest.

In a few seconds, all your file names will be synchronized.
